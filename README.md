## Deploy bot (python 3.7)
1.Set up project's venv

```python3 -m venv venv```


2.Activate the venv

```source ./venv/bin/activate```


3.Install requirements

```pip install -r req.txt```


4.Create .env based on .example-env


5.Run bot

```python bot.py```
